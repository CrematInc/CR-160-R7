The CR-160-R7 board is an evaluation board for Cremat Inc's shaping amplifier 
modules (CR-200-X) and baseline restorer module (CR-210).

Included is a bill of materials (BOM) with supplier information.
The CAD files are included and were created by KiCad.
Gerber files and drill file are also included
This is offered as open source hardware under the MIT license.
